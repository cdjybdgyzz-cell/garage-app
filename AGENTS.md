# AGENTS.md - Garage App

## Project Type
Single-page HTML/CSS/JS app - no build process needed.

## How to Run
Open `index.html` directly in any browser.

## Storage
Data persists in browser's `localStorage` under key `garage_vehicles`.

## Tech Stack
- Pure vanilla HTML, CSS, JavaScript
- No dependencies, no package.json needed
- Mobile-responsive design

## Key Files
- `index.html` - Single file containing all HTML, CSS, and JS
- `package.json` - Empty placeholder (can be ignored)

## No Special Commands
- No lint/typecheck/test commands needed
- No build or dev server required
- Just open index.html in browser

## User Language
German - all UI text is in German.
